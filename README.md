# Villa Agency Contact Page

This project is a simple contact page template for a villa agency website. It is built using HTML, CSS, and Bootstrap framework. The contact page includes:

- A preloader for a better user experience.
- A header with navigation links to other pages.
- A contact form for users to get in touch with agents.
- Contact information including phone number and email.
- An embedded Google Map showing the location of the agency.

## Features

- Responsive design that adapts to various screen sizes.
- Clean and modern UI.
- Easy to customize and integrate into a larger project.

## Usage

To use this template, simply clone the repository and open the `contact.html` file in your web browser.
